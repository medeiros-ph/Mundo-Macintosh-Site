Como fazer backup do Mac?

Vamos utilizar o software nativo da Apple para backup, o Time Machine.

Use o Time Machine, recurso nativo do Mac OS desenvolvido pela Apple para realizar backup dos computadores da Apple, é o melhor recurso para fazer backup automático dos dados pessoais, incluindo apps, músicas, fotos, e-mails, documentos, configurações diversas como e-mail do Mail e boxes do Outlook, biblioteca do Photos até disposição e organização do sistema operacional. Ter um backup permite recuperar arquivos que você excluiu ou retornar versôes do mesmo.

Vamos precisar de um dispositivo para armazenamento, por exemplo: HD externo ou interno, Time Capsule (roteador que possui HD interno) ou similares. Desde que sejam compatíveis com Apple.

Sim, é possível usar vários dispositivos de armazenamento para fazer backup com o Time Machine. Ele consegue gerenciar cronologicamente cada dispositivo. Durante execução do backup, um dispositivo por vez.

Como saber o tamanho apropriado de HD externo?

Verifique quanto de espaço em disco sua máquina posui e quanto tem utilizado...

1° Clique no Menu Apple também conhecido como menu da maçã ();
2° Escolha a opção: "Sobre este Mac";
3° Clique na aba "Armazenamento";

Idealmente, opte por um dispositivo de armazenamento que tenha pelo menos o dobro do espaço total do seu equipamento para usá-lo no processo de backup com o Time Machine.

O dobro do espaço do equipamento pode parecer exagero, porém esse método consiste em guardar seus arquivos no formato de versionamento, quer dizer cronologicamente. Suponhamos que uma vez na semana (durante um ano) você plugue o HD externo na porta USB/Thunderbolt/UBS tipo C do seu Mac e solocite o backup. Você poderá voltar no tempo e verificar o progresso semanal dos seus arquivos desse período de um ano dos seus arquivos de trabalho, pessoais, etc.

O time machine além de guardar os dados como arquivos, guarda seus ajuste e configurações do Mac. Por exemplo, desde a disposição da(s) foto(s) do papel de parede até configuração do Mail, boxes e ajustes do Outlook, etc.

Preparando dispositivo de armazenamento para o backup.

O requisito básico para o Time Machine funcionar, é o dispositivo de armazenamento (HD externo, interno, Time Capsule, etc) estar parctionado como Mac OS Expandido (Journaling)...

Caso o dispositivo contenha qualquer arquivo e não esteja partionado para Apple, guarde todo conteúdo em algum lugar diferente, pois vamos precisar formata-lo na particição correta.

Clique no Spotlight, pesquise por "Utilitário de Disco" e clique nele para iniciar o software (conforme indicação da seta abaixo)...

Disco formatado e pronto para uso...

Caso o processo falhe, repita o procedimento de apagar disco e verifique os ajustes sitados acima. Caso o erro persista, o HD ou disposivo de armazenamento pode ter algum problema de origem física. Então vamos precisar de um novo HD (dispositivo de armazenamento). Antes de descartar o HD, teste com outro cado de dados. Uma das principais causas de falha de disco se trata da idade, pois esses dispositivos tem data de validade ou ainda do cuidado com o mesmo. HDs são extremamente sensíveis e quedas, batidas, movimentos bruscos podem sim causar defeitos permanentes nos mesmos.

Encontrando o Time Machine...

Pode dar "OK" no Utilitário de Disco e encerar o mesmo na bolinha vermelha.

Vamos iniciar a configuração do Time Machine clicando no Menu da maça ( X ), selecionando Preferências do sistema...
e depois Time Machine.

Antes de Selecionar Disco...

Selecione essa opção para sempre que plugar o HD no Mac, o backup comece de forma automática.

Selecione essa opção para que o menu do Time Machine semre fique visível na barra de menus (será muito útil).

Selecionar Disco...

Clique em "Selecionar Disco...", escolha o Disco que formatamos e demos o nome de Time Machine. Depois clique em: "Usar Disco".

Mais de um disco...

Sim, é possível usar mais de um HD para fazer backup com Time Machine, problema nenhum. Mas só é possível utilizar um por vez.

Vale lembrar que o Time Machine exige recursos da máquina, você pode até trabalhar em paralelo, porém a máquina pode apresentar lentidão. Durante o backup a prioridade é o backup.

OBS.: tenha pelo menos 50% de bateria e deixe seu Mac conectado à energia durante esse processo. No caso dos notebooks.

Pronto, agora só aguardar...

O Time Machine vai preparar e fazer o backup agora.
O primeiro backup vai demorar mais, pois é necessário guardar tudo!
Depois é só manter adicionando à sua rotina diária / semanal / mensal, etc.

Menu do Time Machine na barra de menus...

Você pode fechar as preferências do sistema e acompanhar o progresso e status do backup pela barra de menus, clicando no ícone do relógio com a seta no sentido anti-horário.

Nesse mesmo menu, você consegue dar o comando pra iniciar o backup clicando em: "Fazer Backup Agora" ou mesmo o comando para parar o backup clicando em: "Ignorar Este Backup".

The end

Cuide do HD...

Ao término do Backup, ejete o disco com segurança, guade o disco em local seguro, evite quedas, colisões, etc. E não se esqueça de fazer backup com frequencia. Não existe um padrão, pois depende da sua necessidade, mas tenta fazer backup no mínimo uma vez por mês.

Muito importante, o backup do iCloud funciona apenas para as pastas e arquivos que estão nas pastas programadas e devidamente configuradas. E nesse caso não há cobertura dos ajustes, dos aplicativos, do sistema operacional, na migração de todo conteúdo e ajustes para um Mac novo ou provisório e não há como voltar no tempo das versões dos arquivos trabalhados. O backup do iCloud é estático e pronto. O Time Machine cobre muito mais recursos. Eu recomendo a utilização de ambos em conjunto (minha opinião pessoal baseado nos mais de 10 anos de experiência atendendo em domicílio caso por caso). Os dois em conjuntos, o seu backup estará mais que garantido e perfeitamente coberto para eventuais falhas e imprevistos.
