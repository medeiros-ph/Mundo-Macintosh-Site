Últimas versões do Mac OS

2016-2021

Essas versões são as que predominam o mercado atualmente, pois é raro encontrar um Mac anterior a 2007 sendo a máquina principal de trabalho hoje em dia.
Fora as limitações de uso que ocorrem dentro de sistemas anteriores a esses.

Mac OS X Sierra

Versão 10.12 - Setembro de 2016

Mac OS X High Sierra

Versão 10.13.6 - Setembro de 2017
Máquinas de 2011 param nessa versão de sistema operacional.

Mac OS X Mojave

Versão 10.14 - Setembro de 2018

Mac OS X Catalina

Versão 10.15.7 - Outubro de 2019
Máquinas de 2012 tem essa versão de sistema operacional como última disponível.
Exemplo: MacBook Pro (13 polegadas Meados 2012) e iMac (21,5 polegadas, final de 2012 ).

Mac OS X Big Sur

Versão 11.6.5 - Novembro de 2020
Essa versão deu partida aos Macs com os poderosos chips com tecnologia M1, que substituíram os processadores Intel.

Mac OS X Monterey

Versão 12.3.1 - Outubro de 2021
Os Macs compatíveis com ele são:
MacBook (Início de 2016, ou superior), MacBook Air (Início de 2015, ou superior), MacBook Pro (Início de 2015, ou superior),
Mac Mini (Final de 2014 ou superior), iMac Pro (2017) e Mac Pro (Final de 2013, ou superior).

Mac OS X Sonoma

Versão 13 - Setembro de 2023
Última versão disponível até então.
